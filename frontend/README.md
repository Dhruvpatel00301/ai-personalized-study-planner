# Frontend - AI Personalized Study Planner

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Create `.env` from `.env.example`.
3. Start development server:
   ```bash
   npm run dev
   ```
4. Build production bundle:
   ```bash
   npm run build
   ```

## Notes

- Mobile-only UI with max width 430px.
- Desktop/tablet widths show a mobile optimization message.
- Uses Tailwind CSS, React Router, Axios, and Chart.js.
